## PIN CONFIGURATION : 
#### 1)-POTENTIOMETER : 
+V : +5, GND, A0 for Conversion.
#### 2)-Seven Segment : 
a : 10, b : 9, c : 8, d : 7, e : 6, f : 5, g : 4
ControlLowesDigit : 12, ControlHighestDigit : 11

#### 3)-Note : 
Varying the potentiometer will change the value to be displayed based on the voltage that the adc reads.
(0v -> 00, 5v -> 99).
